Abstract

This ontology defines an alignment of the Profile Description Ontology with DCAT through subclassing relevant DCAT types and properties, without limiting the scope of the main Profile Description Ontology to DCAT.

While in development, this alignment is given as RDF data, see link above, and graphically in figures below.
